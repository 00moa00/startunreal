# 프로젝트 소개
Unreal 5.1 학습용으로 제작한 프로젝트입니다. GTFO의 시스템을 일부 모작하였습니다.

# 개발 인원
1인

# 개발 코멘트
언리얼의 기본 시스템 이해를 우선으로 했던 프로젝트입니다. 언리얼의 인터페이스, 컨텐츠 프레임 워크, DataTable과 UI 등 기본적인 기능을 학습할 수 있었습니다.
